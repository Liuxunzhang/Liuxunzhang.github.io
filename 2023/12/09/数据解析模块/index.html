Xpath

xpath是一门XML文档中查找信息的语言，XPath可用来在XML文档中对元素和属性进行遍历，而我们熟知的HTML恰巧属于XML的一个子集，所以完全可以用xpath去查找HTML中的元素

在上述html中

节点的介绍

1，book,id,name,price….都被称作为节点

2，id,name,price,author都被称book的子节点

3，book被id,name,price,author称作为父节点

4，id,name,price,author被称为同胞节点

xpath的运用例子

result = et.xpath(“/book”) #表示根节点

reslut = et.path(“book/name”) #在Xpth中间的/表示是儿子

reslut = et.path(“book/name/text()”)[0] #text()拿文本

reslut = et.path(“book//nick”) # // 表示的是子孙后代

*reslut = et.path(“book//nick/text()”) # 通配符 谁都行*

reslut = et.path(“book/author/nick[@class=”jay”]/text()”) # []表示属性筛选，@属性名 = 值

reslut = et.path(“book/partner/nick/@id”) #最后一个/表示拿到nick里面id的内容，@属性，可以直接拿到属性值

li_list = et.xpath(“html/body/ul/li”) #得到是一个列表

li_list = et.xpath(“html/body/ul/li[]”) #可以在后面直接中括号定位

li_list = et.xpath(“//li”) #可以拿到很多li

for li in li_list:

li.xpath(“./“) #./相对 表示当前的节点，这里的./就是li

li.xpath(“./a/@href”) #想要href ，只写href,不写属性，就是拿取属性

li.xpath(“./a/text()”) #想要文本

pyquery

PyQuery(选择器)

p = PuQuery(html) 将html内容放进PyQuery里面初始化

后代选择器

空格就是后代选择器

.items()

当选择器的内容很多的时候，需要一个一个处理的时候

a = p(“li a”).attr(“href”) 只能拿到第一项数据

多个标签 拿属性，用迭代器,用items()方法

it = p(“li a”).items() 得到一个生成器

attr(属性名):拿到属性的值

加值就是修改或者添加

不加值就是查询

attr可以修改，增加属性

text() 拿到文本
