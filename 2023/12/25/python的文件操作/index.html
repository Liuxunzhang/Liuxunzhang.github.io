python文件操作

打开文件：

open(文件路径,mode=””,encoding=””)

文件路径：

绝对路径 d:/test/xxx.txt

相对路径 相对于当前程序所在的文件夹

../上一层文件夹

打开文件模式mode:

r:read “读取

content = f.read() 全部读取

line = f.readline() 读取一行 但是读取的语句有换行，每句话和每句话之间有空格

line = f.readline().strip() 可以用strip方法去掉字符串两端的空白，如空格、换行、制表符

for line in f:

print(line.strip())从f中读取到每一行数据

w:write “写入”

如果文件不存在，会自动创建文件

每一次open都会清空

f.write(“1”) 写入1，覆写

f.close 每次操作之后要养成好习惯，要关闭连接

f.append(“1”) 追加

f.append(“1”) 追加1

b:读写的是非文本文件

with:上下文，不需要主动取关闭文件
